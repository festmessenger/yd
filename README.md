## Что это

FestMessenger — мессенджер который работает прямо в браузере и хранит все данные на вашем Яндекс Диске. Никаких серверов, никаких подписок. Ваши данные — только ваши.

**Подходит для:**
- Семейного общения
- Корпоративных сетей с белыми списками доменов
- Ситуаций где нужен мессенджер без сторонних серверов

- 🌙 Светлая и тёмная тема

## Быстрый старт

### 1. Получить токен Яндекса

1. Зайдите на [oauth.yandex.ru/client/new](https://oauth.yandex.ru/client/new)
2. Создайте приложение, выберите платформу "Веб-сервисы"
3. Callback URI: `https://oauth.yandex.ru/verification_code`
4. Права: **Яндекс Диск REST API** — все три пункта
5. Скопируйте **ClientID**
6. Откройте в браузере:
   ```
   https://oauth.yandex.ru/authorize?response_type=token&client_id=ВАШ_CLIENT_ID
   ```
7. Разрешите доступ → скопируйте `access_token=...` из URL

### 2. Открыть приложение

Перейдите на https://festmessenger.github.io/yd/ и введите токен.

### 3. Установить как приложение (PWA)

В браузере нажмите "Установить" или "Добавить на рабочий стол".

## Структура данных на Яндекс Диске

```
/BypassChat/
  users/
    ivan/
      profile.json      ← профиль, хэш пароля, настройки
      backgrounds/      ← фоны чатов
  chats/
    dm_ivan_petr/       ← личный чат
      meta.json
      msgs/
        2026-03-15.json ← сообщения за день
      files/            ← прикреплённые файлы
    group_abc123/       ← групповой чат
      meta.json         ← участники и права
      msgs/
      files/
```

## Права администратора

Для входа в панель администратора добавьте в `profile.json` пользователя:
```json
{
  "isAdmin": true
}
```

## Кастомизация

Укажите URL вашего форка в настройках приложения — FestMessenger загрузит темы и конфигурацию из вашего репозитория.
